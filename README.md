# 🌍 AI Global Job Scraper Pro

AI Global Job Scraper Pro is a modern full-stack web application that helps users search job opportunities by role and country. It provides large-scale job listings, filtering options, and downloadable reports for smarter job hunting.

## 🚀 Live Demo

🔗 https://ai-job-scraper-three.vercel.app/job-scraper-pro

## 🚀 Features

- Search jobs by **job title**
- Search jobs in a **specific country**
- Display **100+ job listings**
- Filter by:
  - Remote / Hybrid / On-site
  - Experience level
  - Latest jobs
- Export results as:
  - PDF Report
  - CSV File
- Clean and responsive UI
- Fast and scalable search experience

## 🛠️ Tech Stack

- Next.js
- React.js
- Tailwind CSS
- TypeScript
- API Routes
- jsPDF
- CSV Export
